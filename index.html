EventHub (Event-Driven)

School event registration & notification system

Built with PHP, React, MySQL, Redis, and Docker. A RESTful API for event management with role-based auth (student/organizer), registration with confirmed/waitlist logic, and an async worker for email notifications. Chose event-driven architecture to decouple the API from notification delivery — domain events (RegistrationConfirmed, WaitlistPromoted, EventCancelled) are queued via Redis and processed by a separate worker, keeping the HTTP layer fast and the system extensible.
